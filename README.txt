	#####     Français     #####
Bienvenue dans notre application ! Ce document vous guidera à travers les étapes nécessaires pour installer et utiliser notre programme. Suivez attentivement les instructions pour garantir une expérience optimale.

  ###   Installation des Modules   ###
Avant de commencer, assurez-vous d'installer les modules nécessaires. Ouvrez votre terminal et exécutez les commandes suivantes :

pip install tkinter

pip install customtkinter

  ###   Installation de la Base de Données   ###
Pour installer la base de données, veuillez suivre ces étapes :
1. Ouvrez MySQL Workbench
2. Ajoutez une nouvelle connection
3. Assurez vous que le Username et le Password sont bien tous les deux "root" (où modifiez le code python dans listes.py et resultats.py pour pouvoir établir la connection entre la base de données et le code python)
4. Assurez vous que le logiciel UwAmp vous permet bien de faire la connection 
5. Copiez/Collez le code de la base de données dans MySQL Workbench
6. Lancez le code

  ###   Utilisation   ###
Pour utiliser l'application, veuillez suivre les instructions suivantes :
- Défilement horizontal : Utilisez la combinaison `MAJ + Molette` pour faire défiler horizontalement lors de l'affichage des descriptions des disciplines.
-  Mise à jour de la base de données : Après l'ajout d'un résultat ou d'un sportif, ainsi qu'à la supression d'un sportif, veuillez fermer le programme. Puis relancer le afin de mettre à jour la base de données.

	#####     English     #####

  ### Installing Modules ###
Before you start, make sure you have installed the necessary modules. Open your terminal and run the following commands:

pip install tkinter

pip install customtkinter

  ### Database installation ###
To install the database, please follow these steps:
1. Open MySQL Workbench
2. Add a new connection
3. Make sure Username and Password are both "root" (or modify python code in listes.py and resultats.py to establish connection between database and python code).
4. Make sure that the UwAmp software allows you to make the connection. 
5. Copy/Paste database code into MySQL Workbench
6. Run the code

  ### How to use ###
To use the application, please follow these instructions:
- Horizontal scrolling: Use the `MAJ + Wheel` combination to scroll horizontally when displaying discipline descriptions.
- Updating the database : After adding a result or athlete, or deleting an athlete, please close the program. Then restart it to update the database.
